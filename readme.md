# AllMeshRendererSetter

## 使い方
ヒエラルキーのオブジェクトを右クリックで「AllMeshRendererSetter」が追加されます。

子孫要素のMeshRenderer/SkinnedMeshRendererを一括で設定できます。
[RootBone|Bounds|ProveAnchor]の設定をサポートしています。また、SkinnedMeshRendererのTransformをリセットします。
間違えたらUndoできます。

## インストール
AllMeshRendererSetter.csを適当な場所に保存します。
https://yukiyukivirtual.booth.pm
にあるunitypackageを使ってもいいです。
